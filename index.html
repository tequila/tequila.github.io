    Welcome to Tequila Project¶

    Tequila Project is an organization that creates open-source solutions. We love PHP and MongoDB, therefore our main projects are dedicated to integration of this technologies. Our core projects are:

    • Tequila MongoDB ODM - a simple and very fast MongoDB Object Document Mapper for PHP 5.6 and PHP 7 with Doctrine-like syntax.
    • Tequila MongoDB PHP Lib - a high-level driver for MongoDB, written in PHP.

    There is also integrations for our libraries with PHP frameworks:

    • Tequila MongoDB Pimple service provider - a service provider, which integrates Tequila MongoDB PHP Lib with Pimple Dependency Injection Container and Silex framework.
    • Tequila MongoDB ODM Pimple service provider - a service provider, which integrates Tequila MongoDB ODM with Pimple Dependency Injection Container and Silex framework.
    • Tequila MongoDB ODM Bundle - Symfony Bundle, which integrates Tequila MongoDB ODM with Symfony framework.

    On this site you cand find docs and articles about our projects.
